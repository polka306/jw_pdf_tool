# CLAUDE.md — PDF 편집 툴 협업 규칙

이 파일은 Claude가 이 프로젝트에서 작업할 때 따라야 할 규칙과 맥락을 정의합니다.
변경이 있을 때마다 반드시 이 파일을 업데이트합니다.

---

## 프로젝트 개요

- **목적**: 개인용 PDF 편집 데스크탑 애플리케이션
- **플랫폼**: Windows 11
- **언어**: Python 3.11+
- **GUI**: PyQt6
- **계획서**: `PLAN.md` 참고

---

## 기술 스택

| 역할 | 라이브러리 |
|------|-----------|
| GUI | PyQt6 |
| PDF 렌더링/어노테이션 | PyMuPDF (fitz) |
| PDF 페이지 편집 | pikepdf |
| 이미지 처리 | Pillow |
| PDF 생성 | reportlab |
| 문서 변환 | LibreOffice CLI (별도 설치) |
| 패키지 관리 | uv |

## 커밋 규칙 (필수)

### 커밋 전 체크리스트 — 반드시 순서대로 수행

1. **CHANGE.md 업데이트** — 변경 내용을 최상단에 새 항목으로 추가
2. **README.md 업데이트** — 프로젝트 구조나 사용법이 변경된 경우 반영
3. **CLAUDE.md 업데이트** — 규칙이나 기술 스택 변경이 있으면 반영
4. **위 파일들을 함께 스테이징 후 커밋**

> 문서 업데이트 없이 코드만 커밋하지 않습니다.

### 커밋 단위

- **Phase 단위 커밋**: 각 구현 Phase(1~5) 완료 시 커밋
- **문서 초기화 커밋**: 프로젝트 설정 파일 최초 생성 시 커밋
- 커밋 메시지는 한국어로 작성 가능

### 커밋 메시지 형식

```
[Phase N] 변경 요약 한 줄

- 세부 변경 사항 1
- 세부 변경 사항 2
```

## 코드 규칙

### 파일/모듈 구조
- `app/core/` — 비즈니스 로직 (UI 의존성 없음)
- `app/ui/` — PyQt6 위젯 및 윈도우
- `app/utils/` — 공통 유틸리티
- `main.py` — 앱 진입점만 담당 (로직 없음)

### 코딩 컨벤션
- 함수/변수명: `snake_case`
- 클래스명: `PascalCase`
- 상수: `UPPER_SNAKE_CASE`
- 타입 힌트 사용 권장
- 한국어 주석 허용

### 금지 사항
- `app/core/` 내 PyQt6 import 금지 (UI 분리 유지)
- 전역 변수 사용 금지
- 하드코딩된 파일 경로 사용 금지
