## 1. 문서 관리

### US-001: PDF 파일 열기 (정상)
| 항목 | 내용 |
|------|------|
| 시나리오 ID | US-001 |
| 시나리오 제목 | PDF 파일을 정상적으로 열기 |
| 사전 조건 | 애플리케이션이 실행 중이며, 유효한 PDF 파일이 로컬에 존재 |
| 사용자 행동 | 1. Ctrl+O를 누르거나 파일 > 열기 메뉴 클릭<br>2. 파일 선택 다이얼로그에서 PDF 파일 선택<br>3. 열기 버튼 클릭 |
| 기대 결과 | - 첫 페이지가 뷰어 영역에 렌더링됨<br>- 좌측 썸네일 패널에 모든 페이지 축소판 표시<br>- 상태바에 파일명, "페이지: 1 / N", 줌 배율 표시<br>- 기본 줌 150% 적용 |
| 관련 기능 | 1.1 PDF 열기, 2.1 페이지 렌더링, 3.1 썸네일 표시 |
| 관련 TC | TC-001, TC-006 |
