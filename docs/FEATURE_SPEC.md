# 기능 정의서 — jw_pdf v1.0.0

> 작성일: 2026-04-03  
> 대상 플랫폼: Windows 11  
> 기술 스택: Python 3.11+ / PyQt6 / PyMuPDF / pikepdf

---

## 1. 문서 관리

### 1.1 PDF 열기
| 항목 | 내용 |
|------|------|
| 설명 | 로컬 파일 시스템에서 PDF 파일을 선택하여 열기 |
| 입력 | PDF 파일 경로 (파일 선택 다이얼로그) |
| 동작 | 문서 로드 → 첫 페이지 렌더링 → 썸네일 패널 로드 → 상태바 업데이트 |
| 단축키 | Ctrl+O |
| 제약 | 이전 문서가 열려있으면 자동으로 닫고 새 문서 열기 |

### 1.2 저장 (덮어쓰기)
| 항목 | 내용 |
|------|------|
| 설명 | 현재 문서를 원래 경로에 저장 |
| 동작 | 증분 저장(incremental save) 시도 → 실패 시 전체 저장 폴백 |
| 단축키 | Ctrl+S |
| 제약 | 문서가 열려있을 때만 활성화 |

### 1.3 다른 이름으로 저장
| 항목 | 내용 |
|------|------|
| 설명 | 현재 문서를 새로운 경로에 저장 |
| 동작 | 파일 선택 다이얼로그 → 전체 저장 (가비지 컬렉션 + 압축) |
| 제약 | 확장자 미지정 시 `.pdf` 자동 추가 |

### 1.4 종료
| 항목 | 내용 |
|------|------|
| 설명 | 애플리케이션 종료 |
| 동작 | 열린 문서 닫기 → 윈도우 종료 |

## 2. PDF 뷰어

### 2.1 페이지 렌더링
| 항목 | 내용 |
|------|------|
| 설명 | PDF 페이지를 화면에 표시 |
| 동작 | PyMuPDF로 PNG 렌더링 → QGraphicsScene에 표시 |
| 기본 줌 | 150% |

### 2.2 줌 제어
| 항목 | 내용 |
|------|------|
| 설명 | 페이지 확대/축소 |
| 범위 | 25% ~ 400% |
| 증분 | 15% 단위 |
| 조작 | 줌 인(Ctrl+=), 줌 아웃(Ctrl+-), 화면 맞춤(Ctrl+0), Ctrl+마우스 휠 |
| 표시 | 툴바 스핀박스에 현재 줌 배율 표시 |

### 2.3 페이지 이동 (팬)
| 항목 | 내용 |
|------|------|
| 설명 | 선택 도구 상태에서 페이지 드래그로 스크롤 |
| 조작 | 좌클릭 드래그 (선택 도구 활성 시) |
| 커서 | OpenHand 커서 |

### 2.4 페이지 탐색
| 항목 | 내용 |
|------|------|
| 설명 | 이전/다음 페이지로 이동 |
| 조작 | PgUp(이전), PgDn(다음), 썸네일 클릭 |
| 표시 | 상태바에 "페이지: N / 전체" 표시 |

## 3. 썸네일 패널 (좌측 사이드바)

### 3.1 썸네일 표시
| 항목 | 내용 |
|------|------|
| 설명 | 모든 페이지의 축소판 이미지를 좌측 패널에 표시 |
| 동작 | 백그라운드 스레드에서 비동기 렌더링 (UI 비차단) |
| 크기 | 너비 120px, A4 비율(1:1.5) 유지 |

### 3.2 페이지 선택
| 항목 | 내용 |
|------|------|
| 설명 | 하나 이상의 페이지를 선택 |
| 조작 | 단일 클릭, Ctrl+클릭(추가 선택), Shift+클릭(범위 선택) |
| 동작 | 선택된 페이지가 뷰어에 표시됨 |

### 3.3 컨텍스트 메뉴
| 항목 | 내용 |
|------|------|
| 설명 | 썸네일 우클릭 시 페이지 편집 메뉴 표시 |
| 메뉴 항목 | 선택 페이지 삭제, 선택 페이지 추출, 이 위치 앞에 삽입, 맨 끝에 삽입 |

## 4. 페이지 편집

### 4.1 페이지 순서 변경 (드래그 앤 드롭)
| 항목 | 내용 |
|------|------|
| 설명 | 썸네일 패널에서 드래그하여 페이지 순서 변경 |
| 동작 | 드래그 시작 → 드롭 위치 계산 → MovePageCommand 실행 |
| 실행취소 | 지원 (Ctrl+Z) |

### 4.2 페이지 삭제
| 항목 | 내용 |
|------|------|
| 설명 | 선택한 페이지를 문서에서 삭제 |
| 조작 | Delete 키, 컨텍스트 메뉴, 툴바 버튼, 편집 메뉴 |
| 동작 | 확인 다이얼로그 표시 → 삭제 실행 |
| 제약 | 모든 페이지를 삭제할 수 없음 (최소 1페이지 유지) |
| 실행취소 | 지원 — 삭제된 페이지 스냅샷에서 복원 |

### 4.3 페이지 추출
| 항목 | 내용 |
|------|------|
| 설명 | 선택한 페이지를 새 PDF 파일로 내보내기 |
| 조작 | 컨텍스트 메뉴, 툴바 버튼, 편집 메뉴 |
| 동작 | 저장 경로 선택 → 선택된 페이지로 새 PDF 생성 |
| 출력 | 가비지 컬렉션 + 압축 적용된 PDF |

### 4.4 페이지 삽입
| 항목 | 내용 |
|------|------|
| 설명 | 다른 PDF 파일에서 페이지를 가져와 삽입 |
| 조작 | 컨텍스트 메뉴, 툴바 버튼, 편집 메뉴 |
| 동작 | 삽입 다이얼로그 → 소스 PDF 선택 → 페이지 선택 → 삽입 위치에 추가 |
| UI | 소스 PDF 썸네일 미리보기, 다중 선택 지원 |
| 실행취소 | 지원 — 삽입된 페이지 일괄 삭제로 복원 |

## 5. 어노테이션 (주석)

### 5.1 공통 사항
| 항목 | 내용 |
|------|------|
| 도구 전환 | 툴바 버튼 그룹 (배타적 선택), 어노테이션 메뉴, Escape로 선택 도구 복귀 |
| 색상 | QColorDialog로 RGB 선택 (기본: 빨강) |
| 선 굵기 | 0.5pt ~ 20pt (기본: 2.0pt) |
| 저장 방식 | PDF 콘텐츠 스트림에 직접 기록 → 모든 뷰어에서 표시 |
| 실행취소 | 지원 — 어노테이션 추가 전 페이지 스냅샷으로 복원 |
| 회전 지원 | 페이지 회전(90/180/270°)에 관계없이 정확한 좌표 변환 |

### 5.2 텍스트 도구
| 항목 | 내용 |
|------|------|
| 설명 | PDF 페이지에 텍스트 삽입 |
| 조작 | 클릭 → 텍스트 입력 다이얼로그 → 확인 시 해당 위치에 텍스트 추가 |
| 커서 | IBeam 커서 |
| 폰트 | Helvetica, Times Roman, Courier, 한글(맑은 고딕) |
| 스타일 | 크기(6-72pt), 굵게, 기울임 |
| 한글 지원 | Windows 맑은 고딕 자동 감지 (malgun.ttf / malgunbd.ttf) |

### 5.3 사각형 도구
| 항목 | 내용 |
|------|------|
| 설명 | PDF 페이지에 사각형 그리기 |
| 조작 | 드래그로 영역 지정 → 릴리스 시 사각형 생성 |
| 커서 | Crosshair 커서 |
| 미리보기 | 드래그 중 반투명 프리뷰 표시 |
| 최소 크기 | 3px 미만 드래그는 무시 (오클릭 방지) |

### 5.4 타원 도구
| 항목 | 내용 |
|------|------|
| 설명 | PDF 페이지에 타원 그리기 |
| 조작 | 드래그로 바운딩 박스 지정 → 릴리스 시 타원 생성 |
| 동작 | 사각형 도구와 동일한 인터랙션 |

### 5.5 선 도구
| 항목 | 내용 |
|------|------|
| 설명 | PDF 페이지에 직선 그리기 |
| 조작 | 드래그로 시작점~끝점 지정 → 릴리스 시 직선 생성 |
| 동작 | 색상/선 굵기 적용, 채우기 없음 |

## 7. 문서 변환

### 7.1 이미지 → PDF 변환
| 항목 | 내용 |
|------|------|
| 설명 | 여러 이미지 파일을 하나의 PDF로 합치기 |
| 지원 형식 | JPG, JPEG, PNG, BMP, GIF, TIFF, TIF, WEBP |
| 동작 | 이미지 추가/제거/순서 변경 → 출력 경로 지정 → 변환 |
| 페이지 구성 | 각 이미지가 1페이지 |
| 후처리 | 변환 완료 후 "지금 열겠습니까?" 프롬프트 |

### 7.2 Office 문서 → PDF 변환
| 항목 | 내용 |
|------|------|
| 설명 | Word/Excel/PPT 등 Office 문서를 PDF로 변환 |
| 지원 형식 | DOCX, DOC, XLSX, XLS, PPTX, PPT, ODT, ODS, ODP |
| 의존성 | LibreOffice CLI (별도 설치 필요) |
| 자동 감지 | LIBREOFFICE_PATH 환경변수 → Program Files → PATH 순서로 탐색 |
| 타임아웃 | 120초 |
| 단축키 | Ctrl+Shift+C (변환 다이얼로그 열기) |
| 제약 | LibreOffice 미설치 시 해당 탭 비활성화 (graceful degradation) |

### 7.3 변환 다이얼로그 UI
| 항목 | 내용 |
|------|------|
| 탭 1 | 이미지 → PDF: 파일 추가/제거/위아래 순서, 출력 경로 |
| 탭 2 | 문서 → PDF: LibreOffice 상태 표시, 파일 선택 |
| 진행 표시 | 프로그레스 바 + 상태 메시지 |
| 백그라운드 | 별도 QThread에서 변환 수행 (UI 비차단) |

## 10. 빌드 및 배포

### 10.1 EXE 패키징
| 항목 | 내용 |
|------|------|
| 도구 | PyInstaller |
| 출력 | `dist/jw_pdf-v1.0.0.exe` (단일 실행 파일) |
| 설정 | `pdf_editor.spec` — 콘솔 숨김, 아이콘 포함 |
| 빌드 명령 | `scripts\build.bat` |

### 10.2 개발 환경
| 명령 | 설명 |
|------|------|
| `uv sync` | 의존성 설치 |
| `uv run python main.py` | 앱 실행 |
| `uv run pytest tests/ -v` | 테스트 실행 |

## 11. 비기능 요구사항

| 항목 | 내용 |
|------|------|
| 성능 | 썸네일 백그라운드 렌더링 (UI 비차단), 증분 저장, 어노테이션 후 단일 페이지만 갱신 |
| 안정성 | LibreOffice 미설치 시 graceful degradation, 저장 실패 시 전체 저장 폴백 |
| 호환성 | PDF 표준 어노테이션 — 모든 PDF 뷰어에서 표시 가능 |
| 아키텍처 | Core/UI 분리 (core에 PyQt6 의존성 없음), Signal 기반 느슨한 결합 |
| 확장성 | Command 패턴 기반 Undo/Redo — 새로운 명령 추가 용이 |
