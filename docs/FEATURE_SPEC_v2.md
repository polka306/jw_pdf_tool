## 1. 문서 관리

### 1.1 PDF 열기
| 항목 | 내용 |
|------|------|
| 설명 | 로컬 파일 시스템에서 PDF 파일을 선택하여 열기 |
| 입력 | PDF 파일 경로 (파일 선택 다이얼로그) |
| 동작 | 문서 로드 → 첫 페이지 렌더링 → 썸네일 패널 로드 → 상태바 업데이트 |
| 단축키 | Ctrl+O |
| 제약 | 이전 문서가 열려있으면 자동으로 닫고 새 문서 열기 |
| 상태 | ✅ v1.0.0 완료 |

### 1.2 저장 (덮어쓰기)
| 항목 | 내용 |
|------|------|
| 설명 | 현재 문서를 원래 경로에 저장 |
| 동작 | 증분 저장(incremental save) 시도 → 실패 시 전체 저장 폴백 |
| 단축키 | Ctrl+S |
| 제약 | 문서가 열려있을 때만 활성화 |
| 상태 | ✅ v1.0.0 완료 |

### 1.3 다른 이름으로 저장
| 항목 | 내용 |
|------|------|
| 설명 | 현재 문서를 새로운 경로에 저장 |
| 동작 | 파일 선택 다이얼로그 → 전체 저장 (가비지 컬렉션 + 압축) |
| 제약 | 확장자 미지정 시 `.pdf` 자동 추가 |
| 상태 | ✅ v1.0.0 완료 |

### 1.4 최근 파일 목록 ★
| 항목 | 내용 |
|------|------|
| 설명 | 최근에 열었던 PDF 파일 목록을 메뉴에 표시 |
| 동작 | 파일 → 최근 파일 하위 메뉴에 최근 10개 파일 표시 |
| 저장 | QSettings로 로컬에 저장 |
| 제약 | 파일이 존재하지 않으면 목록에서 회색 처리 또는 제거 |
| 상태 | 📋 v2.0.0 예정 |

### 1.5 종료
| 항목 | 내용 |
|------|------|
| 설명 | 애플리케이션 종료 |
| 동작 | 미저장 변경사항 확인 → 열린 문서 닫기 → 윈도우 종료 |
| 상태 | ✅ v1.0.0 완료 |

### 1.6 인쇄 ★
| 항목 | 내용 |
|------|------|
| 설명 | 현재 열린 PDF를 물리 프린터로 인쇄 |
| 동작 | 인쇄 다이얼로그 → 프린터 선택 → 페이지 범위/매수 지정 → 인쇄 |
| 단축키 | Ctrl+P |
| 옵션 | 전체/현재 페이지/범위 지정, 양면 인쇄, 맞춤/실제 크기 |

## 3. PDF 뷰어 — 시스템 통합 ★ 신규

### 3.1 파일 확장자 연결 (기본 PDF 뷰어 등록)
| 항목 | 내용 |
|------|------|
| 설명 | Windows에서 `.pdf` 파일을 더블클릭하면 본 프로그램으로 열리도록 등록 |
| 동작 | 설정 메뉴 또는 최초 실행 시 "기본 PDF 뷰어로 설정" 옵션 제공 |
| 구현 | Windows 레지스트리에 ProgID 등록 (`HKCU\Software\Classes`) |
| 등록 항목 | `.pdf` 확장자 연결, 아이콘 연결, `shell\open\command` 등록 |
| 해제 | 설정에서 "기본 뷰어 해제" 가능 (레지스트리 항목 복원) |
| 권한 | HKCU 기반 → 관리자 권한 불필요 |
| 상태 | 📋 v2.0.0 예정 |

### 3.2 명령줄 인자 지원
| 항목 | 내용 |
|------|------|
| 설명 | 실행 시 파일 경로를 인자로 전달하면 해당 파일을 즉시 열기 |
| 사용법 | `jw_pdf.exe "C:\path\to\file.pdf"` |
| 동작 | 인자로 전달된 PDF 파일을 자동으로 로드 |
| 다중 인스턴스 | 이미 실행 중이면 기존 인스턴스에 파일 전달 (단일 인스턴스 패턴) |
| 상태 | 📋 v2.0.0 예정 |

### 3.3 드래그 앤 드롭으로 파일 열기
| 항목 | 내용 |
|------|------|
| 설명 | 탐색기에서 PDF 파일을 메인 윈도우에 드래그하여 열기 |
| 동작 | `.pdf` 확장자 검증 → 파일 로드 |
| 피드백 | 드롭 가능 영역 하이라이트 표시 |
| 상태 | 📋 v2.0.0 예정 |

### 3.4 "연결 프로그램" 등록
| 항목 | 내용 |
|------|------|
| 설명 | Windows "연결 프로그램" 목록에 본 프로그램이 나타나도록 등록 |
| 동작 | 설치 시 또는 설정 메뉴에서 등록 |
| 구현 | `HKCU\Software\Classes\Applications\jw_pdf.exe` 등록 |
| 상태 | 📋 v2.0.0 예정 |

### 3.5 시스템 트레이 상주 ★
| 항목 | 내용 |
|------|------|
| 설명 | 앱을 닫아도 시스템 트레이에 상주하여 빠르게 재실행 |
| 동작 | 트레이 아이콘 좌클릭 → 앱 표시, 우클릭 → 최근 파일/종료 메뉴 |
| 옵션 | 설정에서 트레이 상주 ON/OFF 가능 |
| 상태 | 📋 v2.0.0 예정 |

## 4. 가상 PDF 프린터 ★ 신규

### 4.1 가상 프린터 드라이버 설치
| 항목 | 내용 |
|------|------|
| 설명 | Windows에 가상 프린터를 설치하여 모든 프로그램에서 "PDF로 인쇄" 가능 |
| 프린터 이름 | "jw_pdf - PDF Printer" |
| 설치 | 앱 설정 메뉴에서 "가상 프린터 설치" 클릭 또는 설치 마법사에서 선택 |
| 해제 | 설정에서 "가상 프린터 제거" 가능 |
| 권한 | 프린터 드라이버 설치 시 관리자 권한 필요 (UAC 프롬프트) |
| 구현 방식 | Windows Print Spooler API + PostScript/XPS 드라이버 |
| 상태 | 📋 v2.0.0 예정 |

### 4.2 인쇄 워크플로우
| 항목 | 내용 |
|------|------|
| 설명 | 다른 프로그램에서 인쇄 시 PDF 파일로 변환하는 전체 과정 |
| 동작 순서 | 1) 다른 앱에서 인쇄 → 2) "jw_pdf" 프린터 선택 → 3) 스풀 데이터 수신 → 4) PDF 변환 → 5) 저장 다이얼로그 또는 자동 저장 |
| 출력 품질 | 300 DPI 기본, 150/300/600 DPI 선택 가능 |
| 용지 크기 | A4(기본), Letter, Legal, A3, 사용자 정의 |
| 상태 | 📋 v2.0.0 예정 |
